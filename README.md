# user-tasks
docker, node, react, boostrap, mongodb

install 
```
$ npm install
```


run 
```
$ npm start
```


run using docker:
```
$ npm run start:docker
```
stop docker:
```
$ npm run stop:docker
```

run test:
```
$ npm test
```
